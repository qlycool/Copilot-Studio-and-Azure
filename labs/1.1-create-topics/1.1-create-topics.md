## Objectives

- Exercise: Add specific topics of conversation  
    - Task 1: Enable Orchestration based on generative AI

    - Task 3: Test single-turn copilot

## Contents

In this lab, you will perform the actions necessary for building an agent with Copilot Studio for using in the rest of labs.

## Exercise 1: Add specific topics of conversation

### Task 1: Create a single-turn topic

1. To create a new topic, in the upper side of the screen select **‘Topics’** tab, then select **'Add a topic’** and then **‘From blank’** to start creating a new custom topic.

![](./images/image-15.png)

2. In the very first building block of the topic **‘Trigger’** (which is used to describe what the topic does), write the following: ‘This topic answers exclusively questions where price is requested: Can you provide the pricing details for Power Platform licenses?, what is the price for Power Automate premium licenses?, Give the price for a Copilot message in pay as you go?’

![](./images/image-16.png)

3. Select the **‘+’ button** in the middle of the screen to add ‘Send a message’ action

![](./images/image-17.png)

4. Write in the box the following sentence ‘To answer questions related to prices ask a MS commercial representative’

![](./images/image-18.png)

5. Select again the **‘+’ button** to add a ‘End current topic’ action.

![](./images/image-19.png)

6. Select the **‘Save’** button in the upper right corner of the designer, provide a name for the topic and click **‘Save’** button in the dialog.

![](./images/image-20.png)


### Task 2: Test single-turn topic

1. At the box '**Ask a question or describe what you need'** prompt in **Test copilot** pane, type ‘Tell me the price for Power Apps per app licenses’.

![](./images/image-21.png)

Generative orchestration detected a question for ‘Pricing’ topic and played that.

### Task 3: Add a multi-turn topic

1. In the upper side of the screen select **‘Topics’** tab, then select **'Add a topic’** and then **‘From blank’** to start creating again a new custom topic.

![](./images/image-22.png)

2. In the block **‘Trigger’** write the following: ‘This topic answer exclusively questions where contact is requested: I want to contact with a commercial representative, how to contact?’

3. Select the **‘+’ button** in the middle of the screen to add ‘Ask a question’ action

![](./images/image-23.png)

4. Provide the following message in text box: "Please, enter your email address"

5. Click arrow below in **‘Identify’** to choose information to be identified in the answer. Search for email and select it.

![](./images/image-24.png)

6. Click in the three dots upper right corner of the Question action and select **‘Properties’**

![](./images/image-25.png)

7. Click in the box ‘Interruptions’

![](./images/image-26.png)

8. Uncheck **‘Allow switching to another topic’** to avoid jump to another topic if the answer is not an email.

![](./images/image-27.png)

9. Select the **‘+’ button** again in the middle of the screen below ‘Question’ action box to add **‘Send message’** action

![](./images/image-28.png)

10. Write the following in text box: ‘Thanks, a commercial representative will contact to you soon’

![](./images/image-29.png)

11. Finally, as in former single-turn topic, add an **‘End current topic’** action, and **‘Save’** the topic with a name (i.e. **‘Contact’**)

![](./images/image-30.png)

![](./images/image-31.png)

### Task 4: Test multi-turn topic

1. At the box '**Ask a question or describe what you need'** prompt in **Test copilot** pane, type ‘Need to contact’.

![](./images/image-32.png)

Generative orchestration detected now a question to ‘Contact’ topic and played that.
